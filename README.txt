The files contained here are the work done as part  of 'Getting and Cleaning Data Project'

Files Included 
--------------
1. TidyData1.txt
Merges the training and the test sets to create one data set.
Extracts only the measurements on the mean and standard deviation for each measurement.
Uses descriptive activity names to name the activities in the data set
Appropriately labels the data set with descriptive variable names.
2. TidyData2.txt
This is created after performing the above steps.
This is an independent tidy data set with the average of each variable for each activity and each subject.

3. The README.txt that explains the details
4. The Codebook needed for the project. 
5. Run_Analysis.R contains steps performed for getting clean data.
